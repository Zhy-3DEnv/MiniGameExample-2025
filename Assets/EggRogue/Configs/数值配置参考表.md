## 🔧 快速调整指南

### 如果玩家觉得太简单
- 增加怪物血量：+10-20%
- 增加怪物移速：+5-10%
- 增加刷怪频率：+10-20%
- 减少玩家基础属性：-5-10%

- 减少怪物血量：-10-20%
- 减少怪物移速：-5-10%
- 减少刷怪频率：-10-20%
- 增加玩家基础属性：+5-10%

### 如果玩家觉得无聊
- 增加怪物类型多样性
- 增加关卡时长
- 增加特殊事件/机制
- 增加奖励（让玩家有动力）

## 💡 实用公式

### DPS计算
```
玩家DPS = 基础伤害 × 攻击速度 × 伤害倍率
怪物DPS = 怪物伤害 × 攻击频率
```

### TTK（击杀时间）计算
```
TTK = 怪物血量 / 玩家DPS
```

### 难度评估
```
难度 = (怪物数量 × 怪物血量 × 怪物移速) / (玩家DPS × 玩家移速 × 玩家血量)
```

### 经济平衡检查
```
每关获得金币 ≈ 胜利奖励 + (击杀数 × 平均掉落)
应该能购买 ≈ 1-2件商品
```
